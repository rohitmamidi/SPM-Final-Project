### Installig
backend/
```
npm run install
```
client/
```
npm run install
```

### Env Please create .env file and have the following values
backed/.env
```
VITE_GITHUB_CLIENT_ID=
VITE_GOOGLE_CLIENT_ID=
```
client/.env
```
GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=
```

### Running app
backend/
```
npm run dev
```
client/
```
npm run dev
```
