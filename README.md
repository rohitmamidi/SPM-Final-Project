Login app with GitHub

### Front-end
* React js
* [Next UI](https://nextui.org/)
* @octokit/auth
* @react-oauth/google
* react-router-dom

### Back-end
* Express js
* Axios

### Installing
backend/
```
npm run install
```
client/
```
npm run install
```

### Env Please create .env file and have the following values
backend/.env
```
GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=
```
client/.env
```
VITE_GITHUB_CLIENT_ID=
VITE_GOOGLE_CLIENT_ID=
```

### Running app
backend/
```
npm run dev
```
client/
```
npm run dev
```
